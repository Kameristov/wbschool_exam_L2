Что выведет программа? Объяснить вывод программы.

```go
package main

import (
    "fmt"
)

func main() {
    a := [5]int{76, 77, 78, 79, 80}
    var b []int = a[1:4]
    fmt.Println(b)
}
```

Ответ:
```
[77 78 79]

```

Программа выведет с 1 по 3 значения массива `a`, так как `a[1:4]` выбирает элементы с 1 до 4 и инициализирует ими слайс `b`.
